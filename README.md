# SilentShield – A Discreet Emergency Alert System

## Project Overview
SilentShield is a safety application designed to send emergency alerts with the user's live location to predefined contacts during emergencies.

## Features
- One-tap emergency alert
- Live location sharing
- Fast notification to emergency contacts
- User-friendly interface

## Technologies Used
- Android
- Java
- Firebase
- Google Maps API
